Что это?

  • LInfo - сводная информация по устройству/системе

  • vnStat - статистика по сетевым интерфейсам. Настраивается в файле addons/vnstat/config.php (конфиг достаточно прокомментирован (En), разобраться не сложно)

  • phpinfo - информация по php

  • Модули php - вкл./откл. установленных модулей php

  • Управление сервисами - start|stop|restart, редактирование стартовых скриптов, etc

  • OPKG - просмотр, установка/удаление, проверка обновлений. Не работает с интерактивными (требующими взаимодействия с пользователем) пакетами, категорически не рекомендуется обновлять сам пакет "ext-ui-*" (это не баг, это фича)

  • File Manager - файловый менеджер

  • Terminal - SSH доступ через web-браузер (на основе shellinabox)

  • NDMS - "пасхальное яйцо"
